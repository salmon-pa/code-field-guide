How an API actually works

An API (application programming interface) is a waiter between two programs — you don't walk into the kitchen, you give your order to the waiter. Every API call is these four steps:

1

Your code sends a request to a specific address, called an endpoint.

2

It includes a key — proof you're allowed, like showing a membership card.

3

The service does the work and returns an answer, almost always as JSON (labelled data).

4

Your code reads that answer and uses it.

request → key → response → read = every API call

What is MCP

MCP — Model Context Protocol

An AI like Claude or Salmon is smart but starts with no hands — locked in a room with just its brain. MCP is the standard set of hands. It's the plug that lets an AI safely connect to your real tools and data: Gmail, Notion, Google Drive, your calendar.

When your system "uses the Notion MCP," it means the AI now has a safe, standard way to read and write your Notion. It's how your AI team reaches into the real world instead of just talking.

Think: USB-C for AI — one universal plug for connecting AI to tools.

The senior reviewer's cheat sheet

The questions that make your feedback land — ordered by how much they matter.

CriticalSecrets in the code?Keys/passwords belong in env vars or a vault — never in the file. #1 cause of breaches.
CriticalSilent failure on error?A bare except: pass hides bugs and ships wrong data. Loud failure is safe; silent is dangerous.
CriticalUntrusted input trusted?User/API/model input glued into a query or used unchecked → injection. Validate and contain.
HighNo timeout / no retry on a network call?Hangs forever, or dies on a blink. Reliability gap.
HighHardcoded or duplicated?Magic values and copy-paste blocks mean edits in many places. Ask for config / one function.
HighNested loop over big data?Squared cost — fine at 10, dead at 500. Ask about scale.
Worth askingWhere are the tests?No test around important logic is itself a finding.
Worth askingReadable in 6 months?Clear names, a comment on the tricky part. If you can't follow it, that's feedback.
